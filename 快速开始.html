# 快速开始

cooker由引擎和资源文件组成，引擎为单文件jar，资源文件一般指.cooker文件夹。

name url
项目源码 https://github.com/cooker-sast/cooker
项目文档 https://cooker-sast.github.io
realease https://github.com/cooker-sast/cooker/releases
资源文件 https://github.com/cooker-sast/.cooker

# 编译运行（Complie）

推荐用java17编译运行。 如果是编译。先下载项目源码，然后进入项目，运行命令：

mvn clean package
1

在build目录下生成 cooker.jar 文件

也可以直接在realase (opens new window)页面下载编译好的jar

# 资源文件（Resources）

资源文件 (opens new window)内部包含规则文件等，推荐将其放在默认位置$HOME/.cooker。

名称 默认位置 说明 命令行参数 环境变量
资源文件 $HOME/.cooker 存放规则等资源 --resources-dir COOKER_RESOURCE

webvuln 命令的作用是一键web漏洞扫描

以java-sec-code (opens new window)靶场为例（命令详解 中也以该项目为例）。 首先下载靶场代码，然后进行编译，得到 java-sec-code-1.0.0.jar 文件，然后运行cooker：

java -jar cooker.jar webvuln --acp java-sec-code-1.0.0.jar --work-dir vuln --worker 30
