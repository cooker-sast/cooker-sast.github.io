# cooker与neo4j的联动

invoked-chain 和 call-chain 分析会生成 methods.csv 和 relation.csv文件，这两个文件可以导入neo4j当中分析。

  1. 找个目录，启动 neo4j ：
version: '3'
services:
  neo4j:
    container_name: neo4test
    image: neo4j:latest
    ports:
      - "7474:7474"
      - "7687:7687"
    volumes:
      - $PWD/data:/data
      - $PWD/logs:/logs
      - $PWD/conf:/var/lib/neo4j/conf
      - $PWD/import:/var/lib/neo4j/import
    environment:
      - NEO4J_AUTH=neo4j/passssssworddddddd

  1. 搜索 图搜索调用路径 直接找路径，Invoked 是被调用，所以路径的起点是 sink ，路径的终点自定义：
MATCH path = (start:Method)-[:Invoked*]->(end:Method)
WHERE end.class = "com.marstor.msa.sync.web.impl.AxisTransferImpl"
AND start.isSink = "true"
RETURN path
