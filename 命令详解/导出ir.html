                 [--class-feature-yaml-input=<classFeatureFile>]
                 [--class-name=<className>] [--dir-output=<outputDir>]
                 [--resources-dir=<resourcesDir>] [--worker=<worker>]
                 [--world-cache-dir=<worldCacheDir>]

如果 --class-name 和 --class-feature-yaml-input 都为空，则会dump整个acp的所有class gx用的IR是Tai-e IR，而Tai-e IR是基于soot 的 Jimple 设计的。

# 运行（Run）

java -jar gx.jar ir --acp java-sec-code-1.0.0.jar --class-name org.joychou.controller.Rce --dir-output ir
1

导出 org.joychou.controller.Rce 这个类的 ir ，结果放在 ir 文件夹下面

# 结果（Result）

file 说明 case
导出的ir class name作为file name
classes.txt 导出的所有class的name

# --dir-output

name 说明 required default
--dump-ir 导出ir的目录 false "dump-ir"

# --class-name

name 说明 required default
--class-name 导出的类名 false

# --class-feature-yaml-input

name 说明 required default
--class-feature-yaml-input class feature file，用于class的筛选 false

具体格式参考class搜寻
